## ðŸš€ What is TheiaCast?

TheiaCast is a powerful, web-based digital signage solution that allows you to:

- **Manage Multiple Displays** - Control all your screens from one central dashboard
- **Schedule Content** - Create playlists with time windows and day-of-week scheduling
- **Remote Control** - View and interact with displays in real-time
- **Broadcast Messages** - Send urgent messages to all displays instantly
- **Monitor Health** - Track CPU, memory, and disk usage of all devices

## ðŸ’¡ System Requirements

### Server Requirements
- **Operating System**: Linux, Windows, or macOS (Docker required)
- **RAM**: 2GB minimum, 4GB recommended
- **Storage**: 10GB minimum (more for video content)
- **Network**: Reliable internet connection

### Display Device Requirements
- **Operating System**: Windows 10/11 or Linux (Raspberry Pi OS)
- **RAM**: 2GB minimum (4GB for Raspberry Pi 4 recommended)
- **Network**: Stable connection to server
- **Browser**: Chromium-based (automatically installed)
